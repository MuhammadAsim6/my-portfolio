## Technologies Used
- React 18
- Tailwind CSS
- Lucide React Icons

## Setup

1. Install dependencies:
```bash
npm install
```

2. Install Tailwind CSS:
```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

3. Run the development server:
```bash
npm start
```

4. Build for production:
```bash
npm run build
```
